# Intro
Toteutuin web-visualisointi kurssin harjoitustyönä yksinkertaisen portfolio sivupohjan, jossa käytin itselle täysin uutta Bulma frameworkkia.
# Resursointi (ajankäyttö)
Aikaa harjoitustyöhön en käyttänyt ainakaan liikaa. Vähän viimetinkaan lähdin työstämään sivua ja se näkyy myös työn laajuudessa. Aikaa kului lähinnä Bulman eri ominaisuuksien testailuun.
# Teknologiavalinnat
Sivuston toteutukseen käytin Bulma frameworkkia (https://bulma.io). Lähdin kokeilemaan harjoituksen kannalta, jotain muuta kuin kurssin aikana harjoiteltua Bootstrappia. Bulma on 100% responsiivinen ja suuniteltu mobile first mielessä. Bulmalla oli hyvä dokumentaatio omalla sivustollaan, josta oli helppo kokeilla eri ominaisuuksia. Käytin työn pohjana templatea, josta poistin ylimääräisiä ominaisuuksia ja muokkasin elementit omaan järjestykseen ja oman näköisiksi.
# Yleisiä mietteitä työn kulusta ja mitä tuli opittua
Työ jäi turhan suppeaksi, mutta tuli ainakin opittua kuinka nopeasti saa kasaan täysin responsiivisen sivun. Opittua tuli eniten versionhallinnasta ja githubissa työskentelystä, sillä siinä oli muutamia ongelmia mitkä joutui ratkaista stack overflown avuin. Itse sivuston rakentamisesta jäi päällimmäiseksi mieleen frameworkkien käyttöönotto ja sen hyödyntäminen omalla sivulla. Itse otin framworkin käyttöön linkittämällä sen suoraan html tiedostoon cdjns:n kautta. Täytyy vielä tarkemmin tutustua tuohon NPM asennukseen.
# Arvosana-ehdotus itselle
Arvosanaksi työstä antaisin itselleni 2. Ainoana erona opettajanarvostelutaulukon arvosana 2 kohtaan on se, että minun sivustoni skaalautuu hyvin. Siitä huoimatta 2 on mielestäni lähimpänä realistista arvosanaa, koska sivu on sisällöltään hyvin niukka ja templatea on käytetty apuna, vaikka sitä ei suoraan sellaisena olekkaan käytetty.
